En HTML5 no es necesario proporcionar valores para todos los atributos, como los controles, reproduccion automatica y atributos de bucle utilizados con el elemento <video>. Estos atributos son como interruptores de encendido/apagado. si el atributo esta presente, activa esa opcion. Si se omite el atributo, la opcion esta desactivada Si el navegador no admite el elemento <video> o el formato de video utilizado, se mostrara lo que este entre la <video> de apertura y las etiquetas de </video> de cierre.

Preload: este atributo le indica al navegador que hacer cuando se carga la pagina. puede tener uno de tres valores:
none: el navegador no debe cargar el video hasta que el usuario presione Play.
auto: el navegador debe dercargar el video cuando se carga la pagina.
metadata: el navegador debe simplemente recopilar informacion como el tamaño, el primer fotograma, la lista de pistas y la duraion.
