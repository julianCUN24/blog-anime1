Sistemas operativos Microsoft

Sistemas operativos preferidos

Windows XP: Este sistema operativo, que sucedió a Windows 2000 y es el antecesor de Windows Vista, logró convertirse en el más usado del mundo. Cuenta con adaptaciones para diversos entornos, como computadoras domésticas, equipos portátiles y mini-portátiles.

Mi experiencia resumida: Fue el primer sistema operativo con el cual tuve interaccion y con el cual descubri mi gusto por la informatica en mi infancia.

Windows 7: Windows 7 es una versión de Microsoft Windows, línea de sistemas operativos producida por Microsoft Corporation. Esta versión está diseñada para uso en PC, incluyendo equipos de escritorio en hogares y oficinas.

Mi experiencia resumida: Es un sistema operativo muy completo y con el cual comence a realizar soporte tecnico.

Windows 10 es la ultima versión del sistema operativo de Microsoft, que llega para intentar redimir los errores de Windows 8, y hacer la experiencia de uso mucho más familiar y tradicional para el usuario de teclado y ratón, trayendo de regreso varias características que muchos extrañaban, e incorporando nuevas.

Mi experiencia resumida: Es el sistema operativo con el cual mas me he familiarizado y con el cual para es mas sencillo realziarle soporte tecnico.
