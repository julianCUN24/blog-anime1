El elemento <div> permite agrupar un conjunto de elementos juntos en un cuadro de nivel bloque. Por ejemplo, puede crear un elemento <div> para que contenag todos los elementos del encabezado del sitio (el logotipo y la navegacion), o puede crear un elemento <div> para que contenga informacion en bloques.

En un navegador, el contenido del elemento <div> coemzara en una nueva line, pero aparte de esto no hara ninguna diferencia en la presentacion de la pagina. Sin embargar, el uso de un atributoID o Classen en el espacio debe ocupar el elemento <div> en la pantalla y cambiar la pariencia de todos los elementos contenidos en el. Tambien puede facilitar el seguimiento del codigo si ha utilizado elementos <div> para contener cada seccion de la pagina.

Dado que puede haber varios otros elementos dentro de un elemento dentro de un elemento <div>, puede ser util agregar un comentario despues de la etiqueta </div> de cierre. Esto le permite ver claramente a que etiqueta de apertura se supone que corresponda.
